# ChronoMini

ChronoMini est une application web qui simplifie le suivi des heures de garde d'enfants entre assistantes maternelles et parents.

## A quoi ca sert ?

- Pointer l'arrivee et le depart de chaque enfant en un clic
- Suivre les pauses pendant la journee
- Consulter l'historique des presences et exporter les releves en PDF
- Se connecter entre parents et assistantes maternelles pour partager les informations de garde

## Comment ca marche ?

1. Le parent et l'assistante maternelle creent chacun leur compte
2. Ils se connectent entre eux via un code de liaison unique
3. L'assistante maternelle pointe les heures au quotidien
4. Le parent peut consulter les releves a tout moment
5. En fin de mois, chacun peut telecharger un recapitulatif PDF
